# The YHC Way - Recent Changes
## January 7-17, 2026

This document summarizes all major features and improvements made to The YHC Way application over the past 10 days.

---

## January 17, 2026

### Email Signatures
- Added ability to create and manage personal email signatures
- Signatures support rich text formatting (bold, italic, links, etc.)
- Set a default signature that automatically appears when composing or replying to emails
- Manage signatures from Settings page

### Perkville Integration Improvements
- Added diagnostic endpoint for troubleshooting Perkville configuration
- Improved error reporting for authentication issues
- Made integration work reliably in production environments

### AI Email Summarizer
- Added copy button to quickly copy AI-generated email summaries

### Email Templates
- Added rich text editor for creating dynamic email template content

### Task Dependencies
- Added dependency tracking to visualize task relationships
- Dependency tracker page for managing task dependencies

### Slack Direct Messages
- Improved filtering and display logic for Slack DMs
- Added ability to view and manage direct message conversations
- Added DM notification preferences management

### Deleted Emails
- Added ability to view deleted emails within the application
- Added link to Gmail trash

### Email Forwarding
- Added email forwarding functionality

### Slack Channel Configuration
- Improved visibility of Slack channel selection dialog
- Better configuration options for channel preferences

### Sent Email View
- Added 'Sent' view to the Unified Email interface

### Time Tracking
- Notify administrators when users change their time tracker accounts
- Fixed error when linking employee accounts to YHCTime
